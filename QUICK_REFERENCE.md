# FRANVPN – Quick Reference Guide

## One-Page Project Summary

**FRANVPN** is a premium, modern VPN client that combines advanced protocol support with exceptional UX. Built on V2Ray but with complete independent branding and design.

---

## What Makes FRANVPN Different?

✅ **Professional SaaS design** (not "hacker-style")  
✅ **Two user modes:** Simple (beginners) + Advanced (power users)  
✅ **Full protocol flexibility:** VLESS, VMess, Trojan, Shadowsocks, etc.  
✅ **Premium quality:** Commercial-grade UX, not a hobby project  
✅ **Fully independent:** Zero visual association with V2Ray  
✅ **Cross-platform:** Windows, macOS, Linux, iOS, Android  

---

## Core Vision

> FRANVPN is the VPN client for users who demand both security and quality.  
> Modern. Reliable. Transparent.

## Key Features

### For Everyone
- One-click connect/disconnect
- Auto best server (AI-selected)
- Real-time statistics
- Beautiful dark/light themes
- Cross-platform sync

### For Power Users
- Manual protocol selection
- DNS management (DoH, DoT)
- Kill switch + Split tunneling
- Connection logs
- Advanced security settings
- Config import/export
