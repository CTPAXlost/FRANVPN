# FRANVPN – Brand Guidelines

## Brand Identity Foundation

### Brand Purpose

FRANVPN exists to provide users with **secure, transparent, and elegant internet privacy**. We believe privacy shouldn't require sacrificing quality or user experience.

### Brand Mission

> Empower users to protect their digital privacy with confidence, through technology that's both powerful and beautiful.

### Brand Values

1. **Security First** – Privacy is non-negotiable
2. **User Transparency** – Clear communication about what we do and don't do
3. **Design Excellence** – Premium, thoughtful user experiences
4. **Technical Integrity** – No compromises on protocols or performance
5. **Global Accessibility** – Privacy for everyone, regardless of location
6. **Independence** – No corporate tracking or data selling

## Logo Design

### Concept: "The FV Gateway"

The logo represents a gateway or passage – a secure entry point between the user and the internet.

### Logo Variations

#### Primary Logo (Horizontal Lockup)
```
[FV Icon] FRANVPN
```

- **FV Icon:** Stylized geometric gateway/arch shape
- **Wordmark:** "FRANVPN" in sans-serif (Inter Bold)
- **Spacing:** 1.5x the icon width between icon and wordmark

#### Icon Only
```
[FV Icon]
```
- Standalone icon (minimum 32px)
- Can work in app tabs, favicons, etc.

#### Vertical Lockup
```
    [FV Icon]
    FRANVPN
```
- For vertical applications (splash screens, profile pictures)

### Color Variations

**Primary (Recommended):**
- Icon: Gradient (emerald to teal)
- Text: Dark navy (#0F172A) or white (#FFFFFF)

**Monochrome (Light Background):**
- Icon & Text: Dark navy (#0F172A)

**Monochrome (Dark Background):**
- Icon & Text: White (#FFFFFF)

**Inverted:**
- Icon: White
- Text: Dark background

### Clear Space & Sizing

- **Minimum Size:** 32px width (app icon)
- **Clear Space:** 1x the icon width on all sides
- **Never:** Compress horizontally/vertically, rotate, change colors

### Logo Don'ts

❌ Don't use V2Ray logo or assets  
❌ Don't add drop shadows or 3D effects  
❌ Don't use old/different fonts  
❌ Don't modify the icon shape  
❌ Don't use unapproved colors

## Typography System

### Font Stack

**Primary Font:** Inter
- **Source:** Google Fonts (libre font)
- **Fallback:** -apple-system, BlinkMacSystemFont, "Segoe UI", sans-serif

**Why Inter:**
- Modern, clean, geometric
- Excellent readability at all sizes
- Professional SaaS aesthetic
- Strong open-source community

### Typographic Hierarchy

```
Heading 1 (H1)
- Font Size: 32px
- Font Weight: 700 (Bold)
- Line Height: 1.2
- Usage: Page titles, major sections

Heading 2 (H2)
- Font Size: 24px
- Font Weight: 600 (Semi-bold)
- Line Height: 1.3
- Usage: Section headers, card titles

Heading 3 (H3)
- Font Size: 18px
- Font Weight: 600 (Semi-bold)
- Line Height: 1.4
- Usage: Subsection headers, labels

Body (Regular)
- Font Size: 14px
- Font Weight: 400 (Regular)
- Line Height: 1.5
- Letter Spacing: 0.01em
- Usage: Main content, descriptions

Body Small
- Font Size: 12px
- Font Weight: 400 (Regular)
- Line Height: 1.5
- Usage: Secondary text, metadata

Caption
- Font Size: 11px
- Font Weight: 500 (Medium)
- Line Height: 1.4
- Usage: Captions, labels, UI chrome

Code
- Font Family: "Courier New", monospace
- Font Size: 12px
- Font Weight: 400
- Usage: Technical config, error messages
```

### Text Color Guidelines

**Primary Text:**
- Light Mode: #0F172A (Navy)
- Dark Mode: #F1F5F9 (Off-white)

**Secondary Text:**
- Light Mode: #64748B (Gray)
- Dark Mode: #CBD5E1 (Light gray)

**Disabled Text:**
- Light Mode: #AEB8C4 (Muted gray)
- Dark Mode: #64748B (Muted gray)

## Imagery & Illustration

### Photography Style

- **Aesthetic:** Clean, minimalist, modern
- **Subjects:** Global, diverse, confident users
- **Color Grading:** Slight emerald/teal color cast
- **Composition:** Rule of thirds, negative space
- **Resolution:** 300 DPI for print, web-optimized for digital

### Illustration Style

- **Style:** Flat, geometric, modern
- **Color Palette:** Brand colors + neutrals
- **Line Weight:** Consistent 2px strokes
- **Corner Radius:** 4px for consistency
- **Purpose:** Onboarding, empty states, help content

### Icon System

**Style:** Outline style, 2px strokes, 24x24px base size

**Icon Grid:** 24x24px with 4px padding
- Ensures visual consistency
- Clean scaling at different sizes

**Common Icons:**
- Shield (security, VPN active)
- Lock (encryption)
- Globe (servers, worldwide)
- Zap (fast, performance)
- Eye-off (privacy)
- Check (success)
- Warning (alert)
- Settings (configuration)

**Icon Usage:**
- Never scale below 16px
- Never rotate or distort
- Match brand color (emerald primary)
- Consistent stroke weight across all

## Brand Restrictions

### DO ✅

- Use official logo files
- Maintain clear space around logo
- Use brand colors from palette
- Use approved fonts (Inter)
- Keep design modern and clean
- Use emerald for active/success states
- Use consistent icon style

### DON'T ❌

- Don't use V2Ray branding or assets
- Don't rotate, skew, or distort logo
- Don't add drop shadows or effects
- Don't change colors arbitrarily
- Don't use unapproved fonts
- Don't use outdated versions
- Don't use green if it's not emerald
- Don't create custom icon styles
